# Advanced Data Types

- These are the **spatial** and **JSON** data types

## Spatial Data Types

- It is used to store **geometrical and geographical values** such as:
    1. Point
    2. Geometry
    3. Polygon

## JSON Data Types

- It stands for **JavaScript Object Notation**

- It is used to store **key-value** pairs

- Syntax:
    (column name) JSON
